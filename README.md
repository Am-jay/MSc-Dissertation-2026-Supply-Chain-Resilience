# Supply Chain Resilience Analytics - Dissertation Project
**Author:** Jeifa Amoafo  
**Course:** MSc Business Analytics, University of Greenwich
## Description
This repository contains the technical components for my dissertation: "The Role of Supply Chain Resilience Analytics in Mitigating Medical Supply Stockouts."
## Contents
**Data:** `Chapter4_Excel_Appendix_Datasets.xlsx` (Contains Digital, Analytics, Interop, and Governance metrics).
* **Analysis:** `DISSERTATION_PYTHON_NOTEBOOK.ipynb` (Python code for visualization and the prescriptive resilience model).
## Data Sources
Data was compiled from primary organizational assessments and secondary benchmarks (WHO/World Bank) for health systems in England, Ghana, and South Africa (2020-2025).
