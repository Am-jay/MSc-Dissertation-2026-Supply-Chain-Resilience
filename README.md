# Supply Chain Resilience Analytics - Dissertation
**Author:** Jeifa Amoafo  
**Course:** MSc Business Analytics, University of Greenwich
## Description
This repository contains the technical components for my dissertation: "The Role of Supply Chain Resilience Analytics in Mitigating Medical Supply Stockouts."
## Contents
**Data:** `Dissertation Chapter4_Excel_Appendix_Datasets.xlsx` (Contains Digital Coverage, Analytics Class, Interoperabilty, Governance metrics and Stockout Drivers).
* **Analysis:** `Supply Chain Resilence Analytics.ipynb` (Python code for visualization and the prescriptive resilience model).
## Data Sources
Data was compiled from secondary research sources, including international healthcare benchmarks (WHO, World Bank), national health reports, and digital health maturity indices for England, Ghana, and South Africa (2020-2025), all datasets research sources used to compile chapter 4 analysis are also attached in a Zip file as DISSERTATION CHAPTER 4 DATASETS.
## Technical Documentation
The codes documentations and data dictionaries are all documented in the Chapter Seven of my disseration.
